# Direction UI

## Intention

Interface SaaS moderne, claire et premium. Le produit doit sembler robuste pour une direction achat, mais assez simple pour une PME.

## Palette

- fond principal : lavande tres pale ;
- surfaces : blanc et blanc casse ;
- accent principal : violet ;
- texte : aubergine fonce, pas noir pur ;
- statuts : vert, ambre, rouge ;
- bordures : violet tres clair.

## Formes

- cartes principales : 24 a 32 px ;
- boutons : 14 a 18 px ;
- badges : forme pilule ;
- sidebar : grand rayon, effet panneau logiciel moderne ;
- pas de cartes imbriquees inutiles.

## Menu

Le menu doit etre :

- retractable au Sprint 01 ;
- compatible desktop/tablette/mobile ;
- organise par module metier ;
- associe a un switch d'entreprise ;
- lisible sans surcharger l'ecran.

## Animations Sprint 01

Avec Motion :

- transition sidebar ouverte/fermee ;
- apparition des panneaux ;
- selection d'une ligne vers fiche detail ;
- transitions discretes entre sections ;
- respect de `prefers-reduced-motion`.
