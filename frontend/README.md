# Apercu autonome ProcuFlow

Ouvrir `register.html` pour tester le parcours d'inscription ou `index.html` pour acceder directement au tableau de bord. Aucun serveur ni compte externe n'est necessaire.

Fonctions interactives :

- menu retractable sur ordinateur ;
- tiroir de navigation sur mobile ;
- creation de compte en deux etapes ;
- modal de nouvelle demande ;
- confirmations visuelles des actions.

La version applicative complete se trouve dans `apps/web`.
