# AGENTS.md

## Intention

Construire ProcuFlow Africa comme un SaaS professionnel, vendable, multi-tenant et adapte aux realites africaines : fournisseurs, documents, validations, RFQ, bons de commande, factures, paiements et KPI.

## Regles de travail

- Garder le projet cumulatif a chaque sprint.
- Ne jamais mettre de secret DOHONE dans le frontend.
- Privilegier une architecture claire : Laravel API d'un cote, frontend moderne de l'autre.
- Laisser une trace d'audit pour toute action metier importante.
- Garder l'UX dense, lisible, arrondie, premium, avec accents violets.
- Eviter les interfaces noires par defaut ; utiliser un fond clair, ivoire ou lavande tres pale.

## Definition of Done Sprint

- Documentation mise a jour.
- Prototype ou code executable disponible.
- ZIP cumulatif genere.
- Risques et suite logique documentes.
